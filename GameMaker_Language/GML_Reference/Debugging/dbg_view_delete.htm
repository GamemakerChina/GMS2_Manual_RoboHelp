dbg_view_delete

This function deletes an existing debug view created using dbg_view.

 

Syntax:

dbg_view_delete(view);

Argument Type Description
view Debug View Pointer A pointer to a debug view

 

Returns:

N/A

 

Example:

Create Event

debug_view = dbg_view("View", true);
show_debug_overlay(true);

Key Pressed Event - Space

dbg_view_delete(debug_view);

The above code first creates a debug view using dbg_view named "View" in the Create event and stores the pointer to it in an instance variable debug_view. The Debug Overlay is then shown using show_debug_overlay.

In the Space Key Pressed event, the debug view is deleted using dbg_view_delete.
