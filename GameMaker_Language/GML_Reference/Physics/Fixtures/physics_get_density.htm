physics_get_density

When you bind a fixture to an instance using physics_fixture_bind() this returns an "id" for the bound fixture. You can use this id to get the density value of the bound fixture (not the "base" fixture) at any time using this function.

 

Syntax:

physics_get_density(fixture)

ArgumentType Description
fixturePhysics Fixture ID the id of the bound fixture

 

Returns:

Real

 

Example:

var density = physics_get_density(fix_id);
physics_set_density(fix_id, density - 0.1);

The code above gets the current density value for the bound physics properties of the instance and then sets them to a different value.
