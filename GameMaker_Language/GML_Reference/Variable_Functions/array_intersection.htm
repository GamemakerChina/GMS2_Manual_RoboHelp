array_intersection

This function returns a new array with the intersection of all arrays that you provide to it.

The returned array contains the common elements with duplicates removed.

NOTE The order of elements in the new array is not ensured. If you need to sort the array you will need to do so afterwards.

Syntax:

array_intersection(array1, array2 [, array3, ... array_n]);

Argument Type Description
array1 Array The first array
array2 Array The second array
[array3, ... array_n] Array OPTIONAL Additional arrays to include in the intersection

Returns:

Array (the intersection of the provided arrays)

 

Example:

var array1 = [1, 1, 2, 7, 12];
var array2 = [1, 2, 4, 5, 7];
var array3 = [1, 4, 5, 7];

var array_intersection = array_intersection(array1, array2, array3);

The above code first creates three temporary arrays array1, array2 and array3. It then calls array_intersection to create a new array that is the intersection of these three arrays.
