Predicate Function

A predicate function is passed into an array function, such as array_any, along with an array.

GameMaker runs your predicate function on each element of the given array. For each element in the array, it passes two arguments into your predicate function:

  • Value of the array element
  • Index of the array element

In your predicate function, you can use these arguments to calculate a return value. It should return a Boolean, to tell whether the element "qualifies".

Based on what your predicate function returns for each element, either a result value or a modified version of the array is returned. However if you're using one of the _ext functions, the array passed into the function is modified.

Example

Let's create a predicate function to check if an array contains the string "apple" inside it.

For this, we would use the array_any function:

var _array =
[
    "apple",
    "banana",
    "coconut",
    "dragonfruit"
]

var _contains_apple = array_any(_array, function(_val, _ind)
{
    return _val == "apple"
});

show_debug_message(_contains_apple); // prints 1 (true)

Our predicate function checks if _val == "apple" and returns the result. When this is true for any one of the array's elements, array_any returns true.

By default, a predicate function runs on all elements inside the array, however most advanced array functions also take two extra parameters: offset and length.

Offset And Length

The offset is the 0-based index where the function starts operating on the array.

The length is the number of elements that are checked in the array, starting at the offset.

The offset can be a negative number, in which case it will start from the end of the array. For example, an offset of -1 refers to the last element of the array, -2 refers to the second-last element, and so on.

The length can also be negative, in which case the function will operate backwards. For example, if you start with an offset of 3 and a length of 3, then elements 3, 4, 5 will be covered. However with a length of -3, elements 3, 2, 1 will be covered.

If the function returns a modified version of the array, only the elements that were operated on will be returned, in the order they were operated in. For example, with an offset of 3 and length of 3, elements 3, 4, 5 will be operated on and returned as a new array, and the rest of the elements (e.g. 0, 1, 2) will be scrapped.

You can use infinity or -infinity as the length to traverse an array to its end or its beginning, respectively.
