Predicate functions

These are functions that you define in GameMaker and pass as an argument to one of the functions below. You don't execute this function yourself directly, GameMaker executes it for every array element in the range that you provide using the offset and length arguments.

The output is most often a Boolean, as this kind of function usually checks whether an element "qualifies". Sometimes it can return a Any value as well, in cases where the function does a mapping (e.g. array_map).
