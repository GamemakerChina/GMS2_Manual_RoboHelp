array_find_index

This function returns the index of the first element in the input array or in a range of it for which the predicate function returns true or -1 if no such element is found.

Once a value is found the function stops execution so array elements that come after that will not be checked using the predicate function.

 

Syntax:

array_find_index(array, function, [offset], [length]);

Argument Type Description
array Array The array to use
function Function The function to use for each element
offset Real OPTIONAL The offset, or the starting index, in the array. Setting a negative value will count from the end of the array. The starting index will then be array_length(array) + offset.
length Real OPTIONAL The number of elements to traverse. A negative value will traverse the array backwards (i.e. in descending order of indices, e.g. 2 > 1 > 0 instead of 0 > 1 > 2).

Function Syntax:

function(element, index);

Argument Type Description
element Any The current array element
index Real The current array index

Function Returns: 

Boolean (whether the condition checked in the function is true or false)

 

Returns:

Real (the index of the first element found or -1 if nothing was found)

 

Example:

var f = function(element, index)
{
    return (element > 0);
}
var array = [-1, -8, -2, -4, 0, 3, 8, 7, 5];
var index = array_find_index(array, f);

The above code first creates a function that takes an array element and its index as the input. The function returns true if the element is greater than 0 and false if not. An array with values is then created, some negative, others zero or positive.
Finally an index is found using array_find_index and stored in a temporary variable index. This variable will contain the value 5, as the first positive value in the array, 3, is at array index 5.
