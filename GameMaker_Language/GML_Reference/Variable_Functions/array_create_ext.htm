array_create_ext

This function creates an array of the given size and calls the function to initialise each entry.

Using this function you can initialise the array with values that depend on the array index.

The function has to take 1 argument, which is the index, and return a value of type Any for that index, which is the value that is placed in the array at that index.

 

Syntax:

array_create_ext(size, function);

Argument Type Description
size Real The size of the array
function Function The function used to initialize each entry.

Function Syntax:

function(index);

Argument Type Description
index Real The current array index

 

Returns:

Array

 

Example:

var f = function(index)
{
    return index + 1;
}
array = array_create_ext(100, f);
show_debug_message(array);

The above code first creates a temporary function f that takes in an index as the first and only argument and returns that index with 1 added to it. It then uses array_create_ext with this function which creates an array filled with the integer numbers from 1 to 100.
