array_map

This function returns a new array with a modified version of the elements in the array or a range of it.

The new value to be used for every index is given by the predicate function.

 

Syntax:

array_map(array, function, [offset], [length]);

Argument Type Description
array Array The array to use
function Function The function to use for each element
offset Real OPTIONAL The offset, or the starting index, in the array. Setting a negative value will count from the end of the array. The starting index will then be array_length(array) + offset.
length Real OPTIONAL The number of elements to traverse. A negative value will traverse the array backwards (i.e. in descending order of indices, e.g. 2 > 1 > 0 instead of 0 > 1 > 2).

Function Syntax:

function(element, index);

Argument Type Description
element Any The current array element
index Real The current array index

Function Returns: 

Any

 

Returns:

Array

 

Example:

function vowel_or_consonant(element, index)
{
    static vowels = ["a", "e", "i", "o", "u"];
    var is_a_vowel = array_any(vowels, method({"letter": element}, function(element, index) {return (letter == element);}));
    return is_a_vowel ? "vowel" : "consonant";
}
letters = ["a", "b", "c", "d", "e", "f", "g", "h", "i"];
letter_type = array_map(letters, vowel_or_consonant);

The above code first creates a function vowel_or_consonant that checks if the given element (which must be a valid letter) is a vowel. It does that by using array_any. If the letter is found, the function returns the string "vowel", if it is not it returns the string "consonant". It then creates an array letters with the first letters of the alphabet. Finally it calls array_map on the letters array with the vowel_or_consonant function and assigns the result to letter_type. This array will contain the text "vowel" or "consonant", depending on whether the element in the original array was a vowel or a consonant.
