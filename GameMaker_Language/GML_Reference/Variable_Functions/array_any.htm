array_any

This function returns true if the predicate function returns true for any of the elements in the array or in a range of it.

 

Syntax:

array_any(array, function, [offset], [length]);

Argument Type Description
array Array The array to use
function Function The function to use for each element
offset Real OPTIONAL The offset, or the starting index, in the array. Setting a negative value will count from the end of the array. The starting index will then be array_length(array) + offset.
length Real OPTIONAL The number of elements to traverse. A negative value will traverse the array backwards (i.e. in descending order of indices, e.g. 2 > 1 > 0 instead of 0 > 1 > 2).

Function Syntax:

function(element, index);

Argument Type Description
element Any The current array element
index Real The current array index

Function Returns: 

Boolean (whether the condition checked in the function is true or false)

 

Returns:

Boolean (true if there is any element in the array for which the function returns true, false if there isn't any)

 

Example:

instance_data =
[
    {"x": 14, "y": 20, "blend": c_red},
    {"x": 25, "y": 37, "blend": c_teal},
    {"x": 100, "y": 88, "blend": c_lime},
    {"x": 640, "y": 12, "blend": c_orange}
];
var f_check_blend = function(element, index)
{
    return (element.blend == c_red);
}
var any_red_blend = array_any(instance_data, f_check_blend);
show_debug_message("Instance data contains an item with c_red blend colour: " + string(any_red_blend));

The above code first creates an array instance_data. Every array element is a struct that stores a 2D position ("x" and "y") and a "blend" colour. It then creates a predicate function f_check_blend that returns true when the element's "blend" is set to c_red and false if it's not. The code then executes array_any on the array, using the f_check_blend function, and shows a debug message with the result.
