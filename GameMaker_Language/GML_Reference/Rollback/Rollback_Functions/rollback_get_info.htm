rollback_get_info

This function returns a struct containing information about a player. This should only be used when managing player instances manually. If you're using rollback_define_player(), then each player instance will automatically receive this information as variables.

The function takes the ID of the player as an argument, and returns a struct with the following variables:

Variable Type Description
player_avatar_url String URL pointing to the player's avatar image
player_avatar_sprite Sprite Asset Sprite containing the player's avatar
player_name String The name of the player
player_type String Tells whether the player is signed-into GX.games or not

Will be "Guest" for guest users, and "User" for signed-in users
player_user_id String The ID of the player on GX.games
player_prefs Any The preferences set for the player

 

 

Syntax:

rollback_get_info(player_id);

Argument Type Description
player_id Real The ID of the player for which information will be returned.

 

Returns:

Struct
