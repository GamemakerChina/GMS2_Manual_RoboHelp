Rollback Multiplayer

Rollback multiplayer in GameMaker allows you to quickly set up and create a networked multiplayer game for the GX.games target.

This functionality uses GX servers to connect players and synchronise input, so no server set-up is required by the developer to start a new online game.

While the online functionality can only be used with the GX.games target, you can use "Sync Test" to test your multiplayer game locally, with random or mock input being applied to the other players. Sync Test is functional on the Windows and macOS targets in addition to GX.games.
