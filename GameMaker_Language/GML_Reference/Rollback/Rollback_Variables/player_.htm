player_name

This variable stores the name of the current player instance.

This variable only exists in player instances created by rollback_define_player(), and is different for each player.

Guest players will return a placeholder name.

 

Syntax:

player_name;

 

Returns:

String

 

Example:

// Draw event
draw_set_halign(fa_center);
draw_set_valign(fa_middle);

draw_text(x, y - 100, player_name);

draw_set_valign(fa_left);
draw_set_valign(fa_top);

This draws the name of the player 100px above its instance, with a center-middle alignment.
