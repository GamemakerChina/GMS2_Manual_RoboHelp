player_prefs

This variable stores the preferences set for the current player. It's assigned to player instances created using rollback_define_player().

Please read: Player Preferences

 

Syntax:

player_prefs;

 

Returns:

Any

 

Example:

image_index = player_prefs.character;

The above code reads a preference and sets an instance variable in the player.
