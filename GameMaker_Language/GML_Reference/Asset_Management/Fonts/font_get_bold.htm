font_get_bold

With this function you can check any font asset to see if it has the bold flag or not. If it does the function will return true, otherwise it will return false.

 

Syntax:

font_get_bold(ind);

ArgumentType Description
indFont Asset Index of the font to check.

 

Returns:

Boolean

 

Example:

if font_get_bold(fnt_Main)
{
    draw_set_font(fnt_Main);
}

This will set the active drawing font to fnt_Main if it is set as bold in its font properties.
