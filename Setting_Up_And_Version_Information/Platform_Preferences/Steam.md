# Steam Preferences

The Steam SDK is used for Windows, macOS and Ubuntu (Linux) builds that
are to be distributed through the Steam platform. This section contains
the setup options required to get it working within your projects:  
![](/assets/Images/Setup_And_Version/Platform_Preferences/Steam_Prefs.png)  

-   **Path To Steam SDK** : The system path to where you have installed
    the Steam SDK. Once you have set this path to the SDK, you will then
    need to activate Steam in the [Game
    Options](../../Settings/Game_Options) and then for each platform
    add the **App ID** .
